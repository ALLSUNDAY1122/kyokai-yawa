# 長編本文監査 04：家族写真の空席

## 起承転結の確認

### 起

- 毎年の家族写真と一脚の空席を、家族の恒例行事として提示
- 祖母の死と家の売却により、空席を置く習慣が途切れる状況を作った
- 椅子の裏の「冬子」、古い写真の少女、人数の不一致を初期の謎として配置
- 語り手を写真補正の仕事に就く人物とし、怪異を画像劣化や認識ミスとして検証する理由を持たせた

### 承

- 空席が年代ごとにカメラへ近づく変化を、写真を並べる作業から発見
- 顔認識ソフトが少女と空席を語り手本人として分類し、結末の伏線を置いた
- 画像から空席を消すと叔父まで写真と家族の記憶から消える現象を提示
- 元画像の復元と同時に叔父の姿と記憶が戻り、怪異の規則を具体化
- 日記、手紙、新聞記事から、冬子が家族として認められず写真と系譜から消された経緯を明かした

### 転

- 母と大伯父が冬子の存在を否定し、椅子とアルバムの処分を進める
- 冬子を家族でないと断定した直後、母が写真と自分の記憶から消え始める
- 空席なしの撮影で親族が一人ずつ消え、三つ編みの少女がカメラへ近づく
- 語り手が椅子を戻し、冬子の名前と記録を家族へ読み上げる能動的な解決行動を取る
- 家族が名前を口にするたび、消えた人物が写真へ戻る形でテーマと怪異を一致させた

### 結

- 冬子の記録保存と墓への追記により、家族が過去の排除を訂正し始める
- 祖母と冬子が同じ写真へ穏やかに写り、一度は怪異が収まったように見せる
- 新しい空席がカメラ直前へ現れ、顔認識ソフトが語り手を100％一致で分類
- 母が語り手を認識できなくなり、記憶を守る役目が次の犠牲へ変わったことを示す
- 短編版の「今年は私が座ります。次は、写真を見つけた人の番です」を最終文として維持

## 伏線回収

| 伏線 | 回収状況 |
|---|---|
| 祖母が空席の背を撫でる | 冬子の席を毎年確認し、他の家族を守る行為として回収 |
| 椅子の裏の「冬子」 | 家族から消された少女の名前と、存在を戻す鍵として回収 |
| 写真人数が一人多い | 写真には冬子の分が数えられていたことを示す |
| 母が冬子の好物だけ覚えている | 名前は否定しても感情に結びついた記憶が残ることを示す |
| 顔認識ソフトが冬子を真奈と分類 | 最後の空席が真奈の席になる結末へ接続 |
| 空席を消すと叔父も消える | 空席が冬子の代替席であり、無ければ別の人物を奪う規則として回収 |
| 祖母の注意書き | 写真、椅子、名前の三要素が記憶を維持する条件として回収 |
| 空席が毎年カメラへ近づく | 祖母の死後、次に写真を見つける真奈へ近づいていたと判明 |

## 文章監査

- 前作の監視・侵入とは異なり、写真の改変と家族の記憶喪失を恐怖の中心にした
- 冬子を単純な悪霊にせず、排除された人物が自分の席を求める悲劇として描いた
- 家族の差別的な過去を説明だけで終わらせず、現在の否定が新たな消失を起こす構造へ接続した
- 画像補正、顔認識、元画像復元は怪異の規則を確認するために限定し、技術説明を増やしすぎていない
- 人物が消える場面では身体的な残酷描写を避け、写真と記憶の不一致による恐怖を優先した
- 解決後に新しい空席を置き、感動的な和解だけで終わらない結末にした
- 短編版の空席、近づく椅子、祖母の書き置きという核をすべて維持した
