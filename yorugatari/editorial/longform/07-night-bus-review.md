## 維持した要素

### 起

- 障害対応で最終バスを逃した大庭裕也と、妻・息子へ帰ろうとする動機
- 時刻表にない古いバス、反応しないIC端末、異なる年代に見える三人の乗客
- 乗車表へ裕也の氏名が浮かび、乗車時点で運行記録へ登録される導入
- 車内時計が一分戻り、通常の路線と時間から外れたことを示す異変

### 承

- 名前への返事と降車ボタンが、白い列側へ移る意思として扱われる規則
- 高校生・水野莉子と作業着の老人が降車し、反復する停留所の列へ加わる展開
- 赤い傘の女性が過去に降車し、次の便へ再乗車している循環
- 乗車表から、乗客が異なる年に同じ最終便へ迷い込んだことを確認
- 佐久間修一の「乗務継続」と、十年間交代を拒んできた運転手の立場

### 転

- 水守トンネル内で濡れた足音が一席ずつ近づく場面
- 母、妻、息子の声が記憶を利用して返事を誘う構造
- 佐久間も娘の声に呼ばれながら返事を拒む描写
- 自分自身の声による「席を代わって」と、乗車表の「交代待ち」
- 事故当夜の衝突音を再現しつつ、身体的な残酷描写は使用しない

### 結

- 朝の営業所、廃止路線、十年前の事故、死亡した運転手という現実側の記録
- 防犯カメラにはバスが映らず、裕也だけが敷地外から歩いてくる不一致
- 古い乗車券の「交代待ち」が「乗務開始」へ変化
- 車庫が空のまま発車案内とエンジン音が作動
- 電源を切った放送設備から裕也自身の発車放送が流れ、運転手への交代が確定する結末

## 削減した要素

- 各停留所を巡回する回数と、白い列の人数変化の反復
- 事故後の照会手順、警察説明、行方不明者確認の細かな往復
- 同じ規則を再説明する会話と、車内設備の重複描写
- 一年後の再来など、営業所での即時反転後に続く追加場面

## 伏線回収

| 伏線 | 回収 |
|---|---|
| 反応しないIC端末 | 通常の営業便ではなく、乗車表で乗客を管理する事故後の便 |
| 異なる年代の乗客 | 乗車表と行方不明記録から、別々の年に乗車した人物と判明 |
| 名前を読む放送 | 返事を降車意思として扱う怪異の規則 |
| 白い列 | 降車した乗客が次の周回から加わる場所 |
| 佐久間の乗務継続 | 生者との交代を十年間拒み続けた記録 |
| 「席を代わって」 | 裕也を運転席へ移す未来の発車放送へ接続 |
| 交代待ち | 営業所で乗務開始へ変わり、交代が完了 |
