## 起承転結の確認

### 起

- 転職と資金不足を、安い部屋を選ぶ現実的な理由として設定
- 301号室と303号室の間に302号室がない違和感を提示
- 親切な管理人、塗り直された壁、開けたまま使うよう勧められるクローゼットを配置
- `302_たすけて`、黒い接続画面、三回のノックで怪異を開始

### 承

- 午前3時02分にだけ現れるSSIDと警告文を反復
- 分電盤の余分な回路、クローゼット内の強い電波、旧図面で情報を具体化
- 管理人の親切な質問を生活時間の把握へ反転
- 壁内から撮影された歴代住人と主人公の写真を提示
- `302_いったね`から`302_みつかった`へ危険を段階化

### 転

- 合鍵を使った玄関侵入と、クローゼット裏の壁内通路を接続
- 覗き穴、合鍵、生活記録、前居住者の端末で人間による監視事件を確定
- 前居住者・高瀬真帆の失踪と管理人の関与を示唆
- Wi-Fi画面の指示で非常ベルを作動させ、主人公が受動的に助けられるだけでなく脱出行動を取る

### 結

- 管理人の逮捕により人怖として一度解決
- ルーターと端末に電源がなく、五年間動作していない事実で怪異へ反転
- 真帆が302号室ではなく301号室の前居住者だったことを開示
- 新居に`301_たすけて`が現れ、三回のノックと最初の警告文を循環させる

## 伏線回収

| 奥の壁だけ新しく塗られている | 壁内通路への扉を隠した改修として回収 |
| 分電盤の余分な回路 | 消された302号室の設備が残っている証拠として回収 |
| クローゼット内で強い電波 | 壁内に置かれた通信機器へ接続。ただし無電源で怪異を残す |
| 管理人が勤務時間を尋ねる | 侵入しやすい時間帯の把握だったと反転 |
| 三回のノック | 前居住者の警告から、新居まで追う怪異へ接続 |
| 補助錠 | 管理人の合鍵による侵入を一時的に防ぎ、脱出時間を作る |
| `302_みつかった` | 真帆の途切れた動画と主人公の現在の危機を重ねて回収 |

## 文章監査

- Wi-Fi名の変化、図面、写真、通路、端末の順で証拠を強め、真相を一度に説明していない
- 管理人を冒頭から露骨な悪人にせず、親切な行動の意味が後から変わるようにした
- 主人公が安い部屋を手放せない事情を置き、異変からすぐ逃げない理由を補強した
- 技術説明はSSID、電波強度、無電源機器に限定し、専門用語を増やしていない
- 人間犯罪の解決と超常現象の反転を分離し、結末の怪異を弱めない構成にした
- 暴力場面は直接描写を抑え、侵入、記録、足音、壁内空間による恐怖を中心にした
