## 起承転結の確認

### 起

- 機種変更直後に、録音した覚えのない八時間十一分の音声が保存される短編版の導入を維持した
- 語り手・瀬戸真由をコールセンターの研修担当とし、自分の声を大量に録音している理由を設定した
- 復元件数が一件多いこと、録音時間が就寝から目覚まし直前まで一致することを初期伏線として配置した
- 午前三時の鍵、廊下の足音、ベッド脇で止まる気配、「まだ違う」を短編版どおり残した
- 侵入痕跡がないため管理会社と警察へ相談し、防犯カメラを設置する合理的な行動を入れた

### 承

- 防犯映像と別の録音機には異常がなく、新端末が生成する夜間ファイルだけに鍵、足音、第二の声が残る構造にした
- 真由の寝息を数秒遅れて繰り返す第二の呼吸を置き、日ごとに遅れが短くなることで模倣の進行を示した
- 三日目の「もう少し」、四日目の「似てきた」を維持し、声紋だけでなく息継ぎや笑う前の呼吸まで一致させた
- 洗面、コーヒー、改札、会社の入館音、朝の挨拶を順に追加し、録音が侵入記録ではなく翌日の行動練習だと反転させた
- ファイル削除後の「消すと最初からになる」により、音声の中の存在が真由の操作を認識していることを示した
- 端末サポート担当に、八時間分の音声が起床直前の数秒で生成されている矛盾を確認させた

### 転

- 真由が奈緒の家で端末を電源オフにし、眠らない対策を取る展開にした
- 数分の意識欠落だけで六分四十二秒のファイルが生成され、端末ではなく真由の睡眠・意識喪失が条件だと明示した
- 第二の呼吸との時間差をほぼ消し、二人分が一人分に聞こえる段階まで模倣を進めた
- 「起きたら、私のふりをして会社へ行って」を短編版どおり維持した
- 遠くの真由が「私はどこへ行くの」と尋ね、代替者が「夜の方へ」と答えることで、本人が音声側へ移される構造を示した
- 社員証が本人の手元にあるのに入館記録が作られ、会社で真由が目撃されることで、音声内の存在が昼の現実へ出たと示した

### 結

- 会社側の真由は、本人より滑らかで、研修用に録音された理想的な話し方をするようにした
- 電話越しの「まだ起きていたんだ」により、代替者が本人の存在を想定外の残存物として扱う段階へ進めた
- 新しいファイル名を〈夜間用〉とし、本人の声が暗い場所で氏名や会社情報を繰り返す状態へ移した
- 会社へ行った存在と自宅に残る語り手のどちらが現実側の本人かを完全には断定しない
- 最終文「その朝、鏡の中の私は、私が動くよりほんの少し早く笑った。」を維持した

## 伏線回収

| 伏線 | 回収状況 |

| 復元件数が一件多い | 夜間ファイルを生成する未知の保存領域が最初から加わっていた可能性を示した |
| 録音時間が睡眠時間と一致 | 端末操作ではなく、真由の意識がない時間を材料に生成されていた |
| 作成時刻が起床直前 | 実時間の録音ではなく、睡眠時間分が一括生成されていた |
| 鍵の音が真由の帰宅音と同じ | 現実の鍵ではなく、保存済みの生活音を再構成していた |
| 第二の呼吸が遅れて重なる | 真由の呼吸を練習し、日ごとに同期していた |
| 「まだ違う」「もう少し」「似てきた」 | 声、息継ぎ、生活手順の一致度を確認していた |
| 防犯映像に誰も映らない | 初期段階の存在は音声ファイル内にしかいなかった |
| 朝の生活音と会社の音が増える | 翌日の本人として行動するためのリハーサルだった |
| 削除への反応 | ファイル内の存在が真由の操作を認識していた |
| 数分の居眠りで短いファイルができる | 真由の意識喪失自体が生成条件だった |
| 社員証が手元にあるのに入館記録が出る | 模倣完了後、代替者が現実側の生活へ移った |
| 〈夜間用〉 | 本人の方が夜間ファイルへ保存される対象へ変わった |
| 鏡像の先行動作 | 現実側の主導権が本人から代替者へ移り始めた |

## 文章監査

- 短編版の八時間録音、鍵、足音、防犯カメラ、「まだ違う」「もう少し」「似てきた」、本人の声、会社へ行く指示、最後の鏡を維持した
- 異常の進行を、第二の呼吸、声の一致、生活音の追加、職場での目撃という順に段階化した
- 端末、アプリ、音声AIなど特定の製品や技術を原因として断定していない
- 端末設定やセキュリティ回避方法を具体的に説明していない
- 睡眠障害や解離性障害などの医学的診断名を使用していない
- 主人公が警察、端末サポート、同僚へ相談し、単独で異常へ対処し続けない構成にした
- 「既読がつかない」の物理的同居者とは異なり、音声ファイル内で成長する非物理的な代替者を中心にした
- 「宅配ボックスの住人」の建物履歴や管理記録を使わず、波形、呼吸、生成時刻、入館記録を証拠にした
- 「三回ノック」の過去への移動や記録改変ではなく、声と生活手順の模倣による本人置換へ差別化した
- 鏡の異常は結末だけに限定し、長編全体の中心を音声と時間差に保った
