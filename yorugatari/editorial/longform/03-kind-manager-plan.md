# 長編化設計 03：優しい店長

## 目標

- 本文量：日本語で約4,000〜5,000字（約1,100〜1,300語相当）
- 読了時間：約8分
- ジャンル：人怖／職場・監視ホラー
- 中心テーマ：親切によって依存を作る支配、退職しても終わらない雇用関係、善意を装った相互監視
- 読後感：異常な店長一人の事件として終わらせず、元従業員まで参加する「見守り」の仕組みが別の職場へ伸びている余韻を残す

## 登場人物

### 小川春香（語り手・19歳）

大学進学を機に一人暮らしを始めた学生。実家との関係が薄く、生活費を自分で補う必要がある。頼ることを苦手としているが、失敗を責めず助けてくれる店長には徐々に安心感を抱く。

### 高木店長（47歳）

駅前コンビニの店長。新人の失敗を責めず、食事、送迎、急な欠勤にも対応する。従業員を「家族」と呼び、退職者についても生活状況や住所を把握している。本人は監視ではなく、孤立した若者を守る活動だと信じている。

### 榊明日香（元アルバイト・22歳）

春香の前に同じ時間帯で働いていた元従業員。店長の過剰な干渉に気づいて退職したが、その後も元従業員による見守りを受けている。春香へ匿名で警告を送る。

### 松田（元アルバイト・30代）

店長へ強い恩義を感じている元従業員。退職後も「卒業生」として現役従業員の送迎や生活確認を担当する。本人には犯罪行為という認識が薄い。

### 森（同僚・大学生）

店長を心から慕い、親切な職場だと考えている。春香の予定を悪意なく店長へ伝えるため、情報が集まる仕組みの一部になっている。

## 物語の核

高木店長は、孤立しやすい若い従業員を助けるうちに、退職後も生活を管理する独自の「見守り」を作った。従業員から住所、授業予定、交友関係、帰宅経路を集め、元従業員へ地域ごとの確認を依頼している。

一部の参加者は店長への恩義から進んで協力し、一部は借金や知られたくない事情を握られて断れない。店長自身がすべての侵入を行うのではなく、店舗を中心とする複数人の監視網が存在するため、店長が勤務中でも対象者の家へ誰かが入れる。

春香は自分の記録に「合鍵作成済み」とあるのを発見し退職するが、帰宅後、元従業員が室内で食事を用意して待っている。事件後に転居しても、店長の「見守り」は新しい勤務先へ引き継がれる。

## 起承転結

### 起：親切な職場と小さな違和感（約250〜300語相当）

1. 春香は大学進学後の生活費を補うため、駅前コンビニで働き始める。
2. 高木店長はレジミスを責めず、廃棄予定品ではなく自費で食事を買い、急な体調不良にも対応する。
3. 夜勤後は危険だからと自宅まで送る。道中、会ったことのない元従業員の住居を正確に言い当てる。
4. 店長は「辞めても、うちの子であることは変わらない」と説明する。
5. 雇用書類とは別に、緊急連絡先、大学の時間割、よく利用する駅、友人の連絡先を書く「安全確認カード」を渡される。
6. 春香は記入をためらうが、森から全員提出していると聞き、空欄を残して提出する。
7. 翌日、空欄だった友人の名前まで店長が知っている。

#### 起で置く伏線

- 店長が送迎中、退職者の住所と現在の勤務先を覚えている。
- スタッフルームの壁に「卒業生当番」という予定表がある。
- 春香の鍵についた傷が、ある勤務日の後だけ増えている。
- 店長が「一人で困る前に、こちらから気づかないといけない」と繰り返す。

### 承：親切が監視へ変わる（約400〜450語相当）

1. 店長が春香の授業終了時刻、通院、友人との約束を本人より先に把握するようになる。
2. 店外で見知らぬ男女から「店長によろしく」と声をかけられる。全員、元従業員だという。
3. 春香が帰宅経路を変えた夜、店長から「いつもの道でないと心配する」とメッセージが届く。
4. 森は、元従業員が近所を通った際に現在のスタッフを確認する仕組みを「助け合い」だと説明する。
5. 匿名アカウントから「店長の車に乗らないで。鍵を持たせないで」と警告が届く。
6. 春香は榊明日香と接触し、退職後も店長から生活報告を求められたことを聞く。
7. 榊は転居後の住所まで把握され、元従業員が職場や自宅付近へ現れるため、連絡を絶てずにいる。
8. 店長が不在の日、春香はバックヤードの鍵付き棚が開いていることに気づく。
9. 中には退職者を含む従業員別のノートがあり、住所、交友関係、帰宅時間、体調、金銭事情が記録されている。
10. ページ端には「継続」「終了」「再確認」の印がある。「終了」の人物だけ、数年前から連絡先も近況も更新されていない。
11. 春香のページには、部屋の間取り、窓の位置、就寝時間、友人の写真が貼られ、最後に「合鍵作成済み」と記されている。
12. 棚の奥には、部屋番号を書いた複数の鍵と、元従業員へ割り振られた見守り当番表がある。

#### 承で強める心理

- 春香は、店長に助けられた事実があるため、最初から悪意だったと断定できない。
- 同僚や元従業員の一部が本気で「守っている」と考えており、相談相手を選べない孤立感を作る。
- 店長の親切な言葉が、後から行動確認や支配の宣言として意味を変える。
- 監視者が一人ではないため、店長の行動だけを警戒しても安全にならないことを示す。

### 転：退職と室内の見守り（約300〜350語相当）

1. 春香はノートの一部を撮影し、榊と警察相談窓口へ送る。
2. 情報流出を確認するため、森にだけ架空の外出予定を話すと、その場所に元従業員の松田が現れる。
3. 春香は勤務終了後、高木へ退職を告げる。
4. 高木は怒らず、「辞めることと、見守りが必要なくなることは別だ」と穏やかに答える。
5. 「困ったときのため」と食料の入った袋を渡そうとするが、春香は受け取らず店を出る。
6. 帰宅すると、玄関の鍵は掛かっているのに、室内灯がついている。
7. 台所には温められた食事と店のレシートが置かれ、印字時刻は春香が店長へ退職を伝えていた時間である。
8. 店長はその時刻、店舗の防犯映像に映っているため、本人が侵入したのではないと分かる。
9. クローゼットから物音がし、元従業員の松田が現れる。「店長から、今日は一人にしないよう言われた」と説明する。
10. 松田は合鍵で入ったことを悪いと思っておらず、食事をしない春香を心配しただけだと主張する。
11. 春香は玄関へ逃げ、近隣住民へ助けを求める。松田は住居侵入で拘束され、複数の鍵と当番表が押収される。
12. 高木は指示を否定せず、「何か起きてからでは遅い」と同じ言葉を繰り返す。

### 結：終わらない引き継ぎ（約200〜250語相当）

1. 高木と協力者への捜査が始まり、店舗の運営会社は高木を勤務から外す。
2. ただし、従業員が自発的に近況を共有した記録も多く、監視網の全容は明らかにならない。
3. 春香は大学近くの別の地域へ転居し、電話番号とアルバイト先を変える。
4. 数か月間、元の店の関係者は現れず、春香はようやく日常を取り戻す。
5. 新しい職場で、同僚から「前の店長さんが挨拶に来た」と聞かされる。
6. 高木は春香の勤務日と帰宅時間を尋ね、菓子折りと封筒を置いていったという。
7. 封筒には新居近くの地図と、赤字で「再確認」と押された安全確認カードが入っている。
8. その夜、春香のスマートフォンへ、知らない番号から現在地の写真が届く。
9. 写真には新しい職場から帰る春香の背中と、少し離れて歩く新しい同僚が写っている。
10. メッセージは「一人にしないよう、引き継ぎました」で終わる。

#### 最後の一文候補

> 振り返ると、新しい同僚は店で見せたことのない笑顔で、「高木店長って、本当に優しい人ですね」と言った。

## 伏線と回収

| 伏線 | 回収 |
|---|---|
| 元従業員の住所を覚えている | 退職後も生活情報を更新しているため |
| 「卒業生当番」の予定表 | 元従業員が地域ごとに現役・退職者を確認する当番表 |
| 鍵についた新しい傷 | 勤務中に複製のため持ち出された可能性を示す |
| 安全確認カードの空欄が埋まっている | 同僚や元従業員から情報を集約している |
| 店長が勤務中なのに室内へ侵入される | 監視と侵入を担当する協力者が複数いる |
| 「継続」「終了」「再確認」 | 対象者の追跡状態を示し、結末のカードへつながる |
| 食事を与える親切 | 生活管理と室内侵入を正当化する行為へ反転 |

## 執筆上の制約

- 高木を最初から露骨な悪人にせず、実際に従業員を助けた経験と歪んだ使命感を持たせる。
- 監視や侵入の具体的な技術・手口は詳述せず、結果と心理的恐怖を中心にする。
- 元従業員全員を被害者または加害者に固定せず、恩義、恐怖、依存によって立場が異なる集団として描く。
- 前2作品の超常現象とは差別化し、最後まで人間だけで成立する恐怖にする。
- 短編版のノート、合鍵、退職後も見守る台詞、室内のレシートをすべて長編版へ残す。
